# Reusable UI Components from Module 1

## Core Layout Components
- **Navigation**: Header with logo and navigation links
- **ProgressIndicator**: Step-by-step progress visualization with animations
- **SubPageNavigation**: Breadcrumb-style navigation with prev/next buttons

## Shared Components
- **AccordionSection**: Expandable content sections with icons
- **SimulationButton**: Animated button for running simulations
- **WellPlate**: 96-well plate visualization component

## Form & Input Components
- **Range sliders**: For parameter adjustment (bead ratio, elution volume, etc.)
- **Number inputs**: For precise value entry
- **Select dropdowns**: For option selection
- **Edit/Save buttons**: Toggle between view and edit modes

## Visualization Components
- **StandardCurveChart**: Chart.js integration for data visualization
- **PCRProgram**: Temperature cycling visualization
- **SpectrophotometerResults**: NanoDrop-style results display

## Interactive Elements
- **Motion animations**: Framer Motion for smooth transitions
- **Loading states**: Progress bars and spinners
- **Export functionality**: CSV download capabilities

## Design System
- **Color palette**: Primary (blue), Secondary (purple), Accent (cyan)
- **Typography**: Consistent heading hierarchy and text styles
- **Spacing**: 8px grid system
- **Shadows**: Consistent shadow levels for depth
- **Gradients**: Brand-consistent gradient patterns

## Layout Patterns
- **Two-column layouts**: Setup + Preview pattern
- **Card-based design**: White cards with shadows and borders
- **Grid systems**: Responsive grid layouts
- **Full-width sections**: With proper max-width containers
